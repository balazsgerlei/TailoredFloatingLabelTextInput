TailoredTextInputLayout

@IBDesignable open class TailoredTextInputLayout: UIView

A container layout, inherited from UIView, containing a TailoredTextField which highly customizable text input field, inherited from UITextField.

Although it is less convinient to set up than the TailoredTextField itself, this view provide more customization options, for example the drop shadow can be applied to only the TextField, rather than only for all of the components like the floating label, error and detail labels.
